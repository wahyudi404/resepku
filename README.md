## Aplikasi Resepku
### Tata cara instalasi
- clone repository
```
git clone <<link>>
```
- buat database di local server Anda
- copy file .env, setting environment
```
cp .env.example .env
```
- install packages 
```
composer install
```
```
npm install && npm run dev
```
- running aplikasi
```
php artisan serve
```
